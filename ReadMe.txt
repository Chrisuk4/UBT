To run app run "UBT_Main.bat".
In the app there are a few buttons, Do NOT press Killer if do you will need to either restart your computer or log out.

Buttons:

Open CMD: Opens a normal cmd window
Open UBT: Opens a old multipurpose tool i made
Play Snake: Opens a cmd windows with snake game
WIFI: Find all SSID's you have connected to and also their passwords by name
Open PingerToolNoob: Opens a old tool i made for pinging
Open Killer: Kills Explorer wich makes your desktop dissapear, Fix by restarting PC
Open TextToSpeech: Opens a Text to speech window made in vbs
GodMode in C Drive: Gives you a windows folder/tool called the GodMode folder and put it in the C drive (Needs Admin Permissions)
